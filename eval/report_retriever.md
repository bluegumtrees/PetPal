## 7. 结论与简历讲点

- **整体表现**：hybrid+rerank 配置下 hit@1 = 83.3%, hit@3 = 86.7%, hit@5 = 93.3%
- **新 KB（P1.5）召回有效**：新 KB hit@5 = 90.0%（n=20），与原 KB 持平/略有差异
- **rerank 必要性**：hit@5 从 83.3%（无 rerank）→ 93.3%（加 rerank），提升 +10.0%
- **OOK robustness**：OOK top-1 rerank score 均值 = 0.639，vs in-KB 中位 = 0.987，差距 = 0.348
- **延迟**：hybrid+rerank 中位 4829 ms / 均值 5151 ms
